struct TemperDefault{A<:UpdateBool, T<:(Union{AbstractVector{R}, R} where R<:AbstractFloat)}

Default arguments for tempering target function.

Fields

  • adaption::UpdateBool

    Checks if temperature will be updated after new iterations

  • val::Union{AbstractVector{R}, R} where R<:AbstractFloat

    Initial value for temperature if adaption is used. Otherwise used to change shape of temperature until it reaches 1.0. The smaller, the more linear the trajectory. The higher, the sooner 1.0 will be reached.
